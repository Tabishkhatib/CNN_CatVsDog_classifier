****Cat vs Dog Classifier (CNN)**

A deep learning project where I designed, trained, and optimized a Convolutional Neural Network (CNN) to classify images of cats and dogs.

**Project Overview**
This project focuses on building a custom CNN model from scratch using TensorFlow/Keras to accurately classify images. The model i trained was  on the Microsoft Cats vs Dogs dataset and further tested on completely unseen images to validate real-world performance . this project was to understand the fundamentls of deep learning and architechture of CNN

 **Key Features**
Custom multi-layer CNN architecture
Batch Normalization for stable training
Dropout & Spatial Dropout to reduce overfitting
L2 Regularization to penalize large weights
Data Augmentation for improved generalization
Global Average Pooling for efficient feature reduction
EarlyStopping & Learning Rate Scheduling
Performance
Achieved **90% validation accuracy**
Successfully tested on multiple unseen images outside the dataset
Consistent and confident predictions on real-world samples

**Model Testing**
The trained model was:
Saved and reloaded for inference
Tested on external images (not part of training/validation)
Verified visually using prediction outputs and confidence scores
